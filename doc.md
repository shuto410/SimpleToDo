
# Task Manager Documents

## 概要
Todoリストとプロジェクトのタスクを一括で管理するWebアプリ。<br>
ユーザーによるデザインテーマのカスタマイズ性を重視。

## 機能概略
Todoリストとプロジェクトタスク管理はタブ等で切り替え。
* Todoリスト　…　テーマでtodoを分ける。プリセットで買い物や家事、課題など。
